力扣454. 四数相加 II

题目要求：
给出四个list，nums1，nums2，nums3，nums4，从每个数组中取一个数使得：

nums[a]+nums[b]+nums[c]+nums[d]==0nums[a] + nums[b] + nums[c] + nums[d] == 0 nums[a]+nums[b]+nums[c]+nums[d]==0

问有多少个（a,b,c,d）四元组

def fourSumCount(nums1,nums2,nums3,nums4):
    Map = collections.defaultdict(int)
    res = 0
    for m in nums1:
        for n in nums2:
            Map[-(m+n)] += 1
    
    for i in nums3:
        for j in nums4:
            if i+j in Map:
                res += Map[i+j]
    return res

赎金信

力扣383. 赎金信

主要仍然是快速寻找一个元素是否在一个集合里面

题目要求：
给两个字符串：“magazine”和“ransomNote”判断ransomNote能否由magazine中的字符构成
每个字符只能使用一次
例子：
ransomNote: "a" magazine:"b"
return: False

ransomNote: "aa" magazine:"ab"
return: False

ransomNote: "aa" magazine:"aab"
return: True

def canConstruct(ransomNote, magazine):
    doc = [0 for _ in range(26)]
    for l in magazine:
        l_idx = ord(l) - ord("a")
        doc[l_idx] += 1
    
    for l in ransomNote:
        l_idx = ord(l) - ord("a")
        if doc[l_idx] > 0:
            doc[l_idx] -= 1
        else:
            return False
    return True

三数之和

此题非常重要！ 面试原题！@腾讯一面
力扣15.三数之和

题目要求：
给出一个整数数组，需要找到三个数使得：

nums[i]+nums[j]+nums[k]==0nums[i] + nums[j] + nums[k] == 0 nums[i]+nums[j]+nums[k]==0

其中i ≠ j，j ≠ k且i ≠ k
返回和为0且不重复的三元组

此题如果利用哈希表会非常的麻烦，所以最简单的方法是利用双指针
解题逻辑：

  1. 将nums排序
  2. 遍历每个数，在遍历每个数的同时，这个数后面所有的数为一个新数组，利用双指针，从两头往中间根据条件筛选符合条件的left和right
  3. 无论是在外层的循环还是left，right，遇到相同的元素都要跳过，因为题目要求最后的结果中也不能出现重复的三元组
def threeSum(nums):
    res = []
    nums.sort()
    for i in range(len(nums)):
        if i > 0 and nums[i] == nums[i-1]:
            continue
        # 剪枝操作，因为nums已经排序，如果nums[i] >0 
        # 则后面所有的和都会大于0，所以没有所需要的结果
        #### 但是只能用在target==0的情况下 ####
        if nums[i] > 0:
            return res
        left = i + 1
        right = len(nums) - 1
        while left < right:
            if nums[left] + nums[right] > -nums[i]:
                right -= 1
            elif nums[left] + nums[right] < -nums[i]:
                left += 1
            else:
                res.append([nums[i],nums[left],nums[right]])
                # 把相同的相邻元素跳过去
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                # 容易忘记加，记得加
                left += 1
                right -= 1
    return res

四数之和

力扣18.四数之和
与三数之和相似，只不过进阶到了四数之和，并且要求四数之和相加为一个数值target

解题思路：
通过一个for循环把四数之和退阶到三数之和

class solution(object):
    # 重新写一遍三数之和
    def threeSum(self, nums, target):
        res = []
        # 此处不需要sort是因为可以在四数之和的for循环之前sort

        for i in range(len(nums)):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            left = i + 1
            right = len(nums) - 1
            while left < right:
                if nums[left] + nums[right] < target - nums[i]
                    left += 1
                elif nums[left] + nums[right] > target - nums[i]
                    right -= 1
                else:
                    res.append([nums[i],nums[left],nums[right]])
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    left += 1
                    right -= 1
        return res

    def fourSum(self, nums, target):
        res = []
        if len(nums) < 4:
            return res
        nums.sort()
        
        # 只需要循环到倒数第三个元素，因为一共需要4个元素
        for i in range(len(nums)-2):
            # 仍然需要跳过重复的相邻元素
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            # 通过三数之和进行退阶
            result = self.threeSum(nums[i+1:], target - nums[i])
            # 如果有答案就收集，如果没有答案就跳过
            if len(result) == 0:
                continue
            else:
                res.extend([r + [nums[i]] for r in result])
        return res
