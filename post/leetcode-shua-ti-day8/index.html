反转字符串：

力扣344. 反转字符串

对于一个字符串例如: ["h", "e", "l", "l", "o"]
输出：["o", "l", "l", "e", "h"]
要求：使用O(1)的额外空间解决问题

def reverseString(s):
    left = 0
    right = len(s) - 1
    while left <= right:
        s[left], s[right] = s[right], s[left]
        left += 1
        right -= 1
    return s

反转字符串 II

力扣541. 反转字符串

题目要求：
给定一个字符串 s 和一个整数 k，从字符串开头算起，每计数至 2k 个字符，就反转这 2k 字符中的前 k 个字符。

  • 如果剩余字符少于 k 个，则将剩余字符全部反转。
  • 如果剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符，其余字符保持原样。

输入：s="abcdefg", k=2
输出："bacdfeg"

class solution(object):
    def reverseNum(self, nums, start, end):
        left = start
        right = end
        while left <= right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
        return nums
    def reverseStr(self, s, k):
        time = len(s) // (2*k)
        s = list(s)

        for i in range(time):
            s = self.reverseNum(s, i*2*k, i*2*k + k - 1)
        if len(s[time * 2 * k:]) < k:
            s = self.reverseNum(s, time * 2 * k, len(s) - 1)
        elif k <= len(s[time * 2 * k]) < 2*k:
            s = self.reverseNum(s, time * 2 * k, time * 2 * k + k - 1)
        return "".join(s)

def is_num(inputs):
    try:
        if instance(int(inputs),int):
            return True
    except:
        return False
def solution(inputs):
    inputs = list(inputs)

    for i in range(len(inputs)):
        if is_num(inputs[i]):
            inputs[i] = "number"
    return "".join(inputs)

反转字符串里的单词：

力扣151. 反转字符串中的单词

题目要求：
输入：s = "the sky is blue"
输出："blue is sky the"

class solution(object):
    def reverseList(self, l):
        left = 0
        right = len(l) - 1
        while left <= right:
            l[left], l[right] = l[right], l[left]
            left += 1
            right -= 1
        return l
    def reverseWords(s):
        s = s.split()
        s = [word for word in s if word != " "]

        s = self.reverseList(s):
        return " ".join(s)

def reverseList(inputs, start, length):
    left = start
    right = start + length - 1
    while left <= right:
        inputs[left], inputs[right] = inputs[right], inputs[left]
        left += 1
        right -= 1
    return inputs

def solution(input_s, k):
    input_s = list(input_s)
    input_s = reverseList(input_s, 0, len(input_s))
    input_s = reverseList(input_s, 0, k):
    input_s = reverseList(input_s, k, len(input_s) - k)
    return "".join(input_s)
